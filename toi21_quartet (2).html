สรุปโจทย์

ในเมืองนครแห่งหนึ่ง มีสถานที่อยู่ 333 ที่ ได้แก่

  • หมู่บ้าน NNN แห่ง แต่ละแห่งถูกกำกับด้วยตัวเลขตั้งแต่ 111 ถึง NNN
  • ศูนย์ตำบล MMM แห่ง แต่ละศูนย์ถูกกำกับด้วยตัวเลขตั้งแต่ N+1N + 1N+1 ถึง N+MN + MN+M
  • ถนน N+M−1N + M - 1N+M−1 สาย โดยถนนจะไม่มี cycle และ ถนนทุกสายสามารถสัญจรสวนทางกันได้
    โดยในเมืองแห่งนี้ เราจะเรียก “กลุ่มอพยพปลอดภัย” เป็นกลุ่มของหมู่บ้าน 444 แห่ง ที่จะต้องมีศูนย์ตำบลอย่างน้อย 111 ศูนย์ ที่ชาวบ้านจากทั้ง 444 หมู่บ้านสามารถเดินทางมายังศูนย์ตำบลศูนย์นั้นได้โดยไม่ต้องใช้ถนนสายเดียวกันเลย

สิ่งที่ต้องทำ

นับจำนวน “กลุ่มอพยพปลอดภัย” โดยให้ระบุคำตอบในรูปแบบของเศษจากการหารจำนวนของ “กลุ่มอพยพปลอดภัย” ด้วย 109+710^9 + 7109+7

ขอบเขตข้อมูล

444 ≤\leq≤ NNN ≤\leq≤ 10510^5105
111 ≤\leq≤ MMM ≤\leq≤ N−2N - 2N−2

ข้อสังเกต

จำนวนถนน จะเท่ากับ N+M−1N + M - 1N+M−1 ซึ่งหมายความว่า Graph ที่ได้มาจะมีลักษณะเป็น Tree

ไอเดียหลัก

เราจะเก็บว่า สำหรับแต่ละศูนย์ตำบล uuu ถ้าเราเลือกให้หมู่บ้าน 444 แห่งเดินทางมายังศูนย์ตำบล uuu โดยไม่ใช้ถนนสายเดียวกันเลย เราจะนับได้กี่กลุ่ม (นับจำนวนกลุ่มอพยพปลอดภัยที่สามารถเดินทางมา uuu ได้)

Prerequisites

  • DFS (Depth First Search)
  • Dynamic Programming
    • Knapsack DP
    • DP on Tree

วิธีทำ

เราจะทำ Dynamic Programming on Tree บน Tree ที่ได้รับมา โดยเราจะเก็บใน array of vector (ตั้งชื่อว่า cntcntcnt) ว่า ถ้าเราอยู่ที่โหนด uuu แล้ว หากเราเดินตามถนนแต่ละสายที่ติดกับโหนด uuu (เดินตาม adjacency list) แล้ว จะมีหมู่บ้านจำนวนกี่หมู่บ้านในเส้นทางนั้น (นับจำนวนหมู่บ้านที่อยู่ใน subtree ของ adj[u][i]adj[u][i]adj[u][i] แล้วเก็บไว้ใน cnt[u][i]cnt[u][i]cnt[u][i]) นั่นคือ ขนาดของ cnt[u]cnt[u]cnt[u] สำหรับแต่ละ uuu อาจจะมีขนาดไม่เท่ากันนั่นเอง
เมื่อเราได้ cntcntcnt มาแล้ว ทีนี้ เราจะทำการ loop จาก N+1N + 1N+1 ถึง N+MN + MN+M (loop แค่ศูนย์ตำบล) แล้วจะคำนวณว่า ถ้าหากว่าเราจะนับจากศูนย์ตำบลที่ uuu แล้ว จะมี “กลุ่มอพยพปลอดภัย” ทั้งหมดกี่กลุ่ม และเมื่อเราคำนวณครบสำหรับศูนย์ตำบลทั้ง MMM ศูนย์แล้ว เราจะได้คำตอบออกมา
วิธีการคำนวณ:

  • การคำนวณแบบ Brute Force: เราจะทำการ loop 444 ชั้น สำหรับทุกๆ uuu โดยจะ loop i,j,k,li, j, k, li,j,k,l และสำหรับทุกๆครั้งที่ loop เราจะเพิ่มค่า ansansans ไปเป็น cnt[u][i]∗cnt[u][j]∗cnt[u][k]∗cnt[u][l]cnt[u][i] * cnt[u][j] * cnt[u][k] * cnt[u][l]cnt[u][i]∗cnt[u][j]∗cnt[u][k]∗cnt[u][l] แต่วิธีนี้จะช้าเกินไปเพราะในเวลาที่แย่ที่สุดจะต้องใช้ time complexity ในการคำนวณ O(n4)O(n^4)O(n4)
  • การคำนวณโดยใช้ Dynamic Programming: เราจะมี 2d array ตั้งชื่อว่า dpdpdp โดยการคำนวณจะเป็นดังนี้:
    • State ของ Dynamic Programming จะเป็น dp[i][j]=dp[i][j] =dp[i][j]= จำนวนวิธีที่จะเลือกหมู่บ้านมา jjj หมู่บ้าน ถ้าหากว่าเราคิดแค่ iii เมืองแรกที่เชื่อมติดกับโหนด uuu
    • Recurrence Relation จะเป็น:
      • dp[i][0]=1dp[i][0] = 1dp[i][0]=1 (มีวิธีเลือก 000 เมืองทั้งหมด 111 วิธี ก็คือ ไม่เลือกอะไรเลย)
      • dp[i][j]=dp[i−1][j]+(dp[i−1][j−1]∗cnt[u][i])dp[i][j] = dp[i - 1][j] + (dp[i - 1][j - 1] * cnt[u][i])dp[i][j]=dp[i−1][j]+(dp[i−1][j−1]∗cnt[u][i]) (จะมี 2 กรณีได้แก่: 1. ไม่เลือกจาก adj[u][i]adj[u][i]adj[u][i] และ 2. เลือกจาก adj[u][i]adj[u][i]adj[u][i])
    • แล้วตอนจบ เราก็แค่เพิ่ม ansansans ไปกับค่าของ dp[adj[u].size()][4]dp[adj[u].size()][4]dp[adj[u].size()][4]

Summary

  • ใช้ DP on Tree ในการคำนวณหาขนาดของแต่ละ subtree ของแต่ละ node
  • ใช้ DP ในการคำนวณวิธีการเลือกหมู่บ้าน 444 แห่งจากศูนย์ตำบลแต่ละศูนย์

Solution Code:

#include <bits/stdc++.h> 

using namespace std; 

#define int long long 

const int mod = 1e9 + 7; 
const int N = 1e5 + 5; 
const int M = 1e5 + 5; 

int n, m, ans, sz[N + M]; 
vector <int> adj[N + M], cnt[N + M]; 

void dfs(int u, int p){ 
	sz[u] = (u <= n); 
	for (auto v : adj[u]) { 
		if (v == p) continue; 
		dfs(v, u); 
		if (sz[v] > 0) cnt[u].emplace_back(sz[v]), sz[u] += sz[v]; 
	} 
	cnt[u].emplace_back(n - sz[u]); 
} 

int32_t main(){ 
	cin.tie(NULL)->sync_with_stdio(false); 
	cin >> n >> m; 
	for (int i = 1; i < n + m; i++) { 
		int u, v; cin >> u >> v; 
		adj[u].emplace_back(v); 
		adj[v].emplace_back(u); 
	} 
	dfs(1, 1); 
	for (int u = n + 1; u <= n + m; u++) { 
		int siz = adj[u].size(); 
		if (siz >= 4) { 
			int dp[siz + 1][5]; 
			memset(dp, 0, sizeof dp); 
			dp[0][0] = 1; 
			for (int i = 1; i <= siz; i++) { 
				dp[i][0] = 1; 
				for (int j = 1; j <= 4; j++) { 
					dp[i][j] = dp[i - 1][j] + (dp[i - 1][j - 1] * cnt[u][i - 1]); 
				} 
			} 
			ans += dp[siz][4]; ans %= mod; 
		} 
	} 
	cout << ans; 
}

Total Time Complexity: O(n+m)O(n + m)O(n+m)
